# การสร้าง pop up
- สามารถใส่ role="dialog" เข้าไปเพื่อเเจ้งให้กับผู้ใช้เเละ ย้าย focus มาอยู่ในพื้นที่ popup
- aria-modal="true" เหมือนเป็น status บอกอีกทีว่า dialog เปิดอยู่ 
    - ไม่ได้มีส่วนในการเปิดปิด popup -> ใช้เป็นในส่วน code เปิดปิดเอง
